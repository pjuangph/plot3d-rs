Expand description

Merge all discoverable n×n×n cube groupings using connectivity data.

connectivities should be the face matches returned by connectivity/ connectivity_fast. The routine builds a graph, performs BFS to locate candidate cube groupings (cube_size^3 nodes), merges their blocks using combine_blocks_mixed_pairs, and keeps track of which original indices fed each merged component.
