Utilities for detecting rotational periodicity in structured multi-block grids.

This module mirrors the behaviour of the original Python tooling and is covered end-to-end by the integration test in tests/test_rotational_periodicity.rs. Generate HTML documentation with cargo doc --open to browse rendered versions of these notes alongside the Rust API surface.
