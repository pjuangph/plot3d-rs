) -> Option<(usize, usize)>
Expand description

Find the indices of the closest point in a 2D face grid to (x, y, z).

The face grid is represented by flat arrays x2, y2, z2 of length nu * nv, stored row-major (u varies fastest).

Returns Some((u, v)) if the minimum distance is within tol, otherwise None.
